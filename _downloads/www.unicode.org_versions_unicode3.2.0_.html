Unicode 3.2.0

Version 3.2.0 has been superseded by the latest version of the Unicode Standard.

Version 3.2.0 of the Unicode Standard consists of Unicode Standard Annex #28, Unicode 3.2, the delta code charts for this version, the Unicode Standard Annexes, and the 3.2 Update of the Unicode Character Database (UCD).

A complete specification of the contributory files for Unicode 3.2.0 is found on the page Components for 3.2.0. That page also provides the recommended reference format for this version of the Unicode Standard.
