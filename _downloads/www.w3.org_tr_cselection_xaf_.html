Abstract

This document specifies a set of XPath functions that can be used to manipulate the Delivery Context associated with a request for an item of content. These functions have been designed to satisfy the requirements to adapt content based on the Delivery Context. While designed to work with Device Independent Content Selection it can be used in any XPath processor.

There is a companion document which provides introductory information concerned with this specification, Content Selection for Device Independence Primer.
