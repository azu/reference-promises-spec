Abstract

This document defines platform and language neutral programming interfaces that provide Web applications access to a hierarchy of dynamic properties representing device capabilities, configurations, user preferences and environmental conditions.

Status of this Document

This document has been discontinued as part of the closure the Ubiquitous Web Applications Working Group.

Readers of this document may also be interested in the work of the Device APIs and Policy Working Group. Those interested in the location example may also be interested in the Geolocation Working Group.
