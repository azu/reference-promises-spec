Latest “RDF Primer” versions

This page (http://www.w3.org/TR/rdf-primer/) contains links to the “RDF 1.1 Primer” W3C Working Group Note (published in 2014, referred to hereafter as “RDF 1.1”) and the “RDF Primer” W3C Recommendation (published in 2004, referred to hereafter as “RDF 1.0”).

RDF 1.1 supersedes RDF 1.0; application developers and implementers of tools should use RDF 1.1. A detailed overview of the differences between the two versions can be found in “What’s New in RDF 1.1”.

The exact references of the two documents are as follows:

RDF 1.1 Primer
W3C Working Group Note 24 June 2014
http://www.w3.org/TR/2014/NOTE-rdf11-primer-20140624/
RDF Primer
W3C Recommendation 10 February 2004
http://www.w3.org/TR/2004/REC-rdf-primer-20040210/
