Latest “RDF Semantics” versions

This page (http://www.w3.org/TR/rdf-mt/) contains links to the “RDF 1.1 Semantics” (published in 2014, referred to hereafter as “RDF 1.1”) and the “RDF Semantics” (published in 2004, referred to hereafter as “RDF 1.0”) W3C Recommendations.

RDF 1.1 supersedes RDF 1.0; application developers and implementers of tools should use RDF 1.1. A detailed overview of the differences between the two versions can be found in “What’s New in RDF 1.1”.

The exact references of the two documents are as follows:

RDF 1.1 Semantics
W3C Recommendation 25 February 2014
http://www.w3.org/TR/2014/REC-rdf11-mt-20140225/
RDF Semantics
W3C Recommendation 10 February 2004
http://www.w3.org/TR/2004/REC-rdf-mt-20040210/
